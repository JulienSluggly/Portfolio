Projets d'études

2023 - Application d'optimisation de graphes

Pour la participation aux Graph Drawing Challenge de 2022 et 2023, j'ai développé un outil permettant de visualiser et d'exécuter divers algorithmes d'optimisation sur les graphes.
Celui-ci est programmé en C++ et utilise OpenGL. Divers algorithmes ont été développés en utilisant le multi-threading ainsi que l'utilisation de CUDA.


2022 - Projet de réalité augmentée

J’ai utilisé OpenCV pour développer une application C++ permettant à un utilisateur ayant un téléphone portable d'interagir avec une construction holographique.
L’application représentait un digicode et a été développée dans le cadre de prévention contre la covid-19.

2022 - Application QR Code

J’ai développé un programme utilisant Cordova, compatible navigateur et Android, permettant de générer et de lire des QR Codes personnalisés via une capture d’écran ou un scan avec un téléphone.

2021 - Application de simulation de communication véhicule à véhicule

J'ai développé en C++ avec Qt Creator une application permettant de simuler les déplacements et la communication de véhicule à véhicule dans la ville de Mulhouse en utilisant les données d'OpenStreetMap.

2019-2020 - Projet Intégrateur

Durant ma dernière année de licence à l’université, j’ai participé au développement d’un jeu vidéo adapté du jeu de société Sushi Go!
J’ai développé la partie backend serveur en utilisant Node.js pour permettre de communiquer avec des clients sur différentes plateformes : Navigateur Web et Client Unity.
J’étais également chargé du traitement des messages du serveur sur les différents clients ainsi que de la mise en œuvre de mesures de sécurité pour éviter la triche, le brute force ou l’injection de données.

Projets Personnels

Wowdle - Jeu navigateur

J'ai développé un jeu sur navigateur basé sur le fameux Wordle.
Chaque jour, un personnage du jeu World of Warcraft est tiré au hasard et le but du jeu est de deviner le personnage à partir d'indices.
Le jeu est développé en utilisant les langages HTML, CSS et JavaScript.
Le jeu est disponible à cette adresse : https://sluggly.github.io/Wowdle

Mod Minecraft - Time Mercenaries

Le jeu Minecraft permet le développement de mods avec l'utilisation de modloaders.
En utilisant Minecraft Forge Modloader, j'ai développé en Java le mod Time Mercenaries pour la version 1.20.1 du jeu.
Ce mod est disponible sur Curseforge qui répertorie une grande partie des mods du jeu.

MedianXL File Manager - Application de gestion de fichiers

J'ai développé en utilisant le C++ avec Qt Creator une application permettant de gérer les fichiers du jeu Median XL.
Cet outil permet de rapidement gérer la liste de personnages (ajout/suppression) et permet de faire des backups.
